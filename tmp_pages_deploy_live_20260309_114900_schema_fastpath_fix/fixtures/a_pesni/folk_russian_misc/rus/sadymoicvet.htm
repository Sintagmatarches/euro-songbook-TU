САДЫ МОИ, САДЫ ЦВЕТИКИ

Сады мои, сады цветики,
Сады цветики лазоревые,
Ой ли, ой люли, ой люли,
Сады цветики лазоревые.

Сады цветики лазоревые,
Соловьи поют веселые,
Ой ли, ой люли, ой люли,
Соловьи поют веселые.

Понагнулися две веточки на двор,
Сокатилися два яблочка долой,
Ой ли, ой люли, ой люли,
Сокатилися два яблочка долой.

Два яблочка, два садовые,
Два садовые медовые,
Ой ли, ой люли, ой люли,
Два садовые медовые.

Два наливчаты, рассыпчатые,
Два по блюдечку катаются,
Ой ли, ой люли, ой люли,
Два по блюдечку катаются.

Два по блюдечку катаются,
Словно сахар, рассыпаются,
Ой ли, ой люли, ой люли,
Словно сахар, рассыпаются.

Словно сахар, рассыпаются,
Виноградом озываются,
Ой ли, ой люли, ой люли,
Виноградом озываются.

Виноградик в саду, яблочек,
Что Иван-то Степанович,
Ой ли, ой люли, ой люли,
Что Иван-то Степанович.

Ему душечку хорошею,
Что и Марью Ивановну,
Ой ли, ой люли, ой люли,
Что и Марью Ивановну.




Русские народные песни. Песенник. Выпуск 4. Составитель Юрий Зацарный. М., «Советский композитор», 1982.
