ВНИЗ ПО МАТУШКЕ ПО ВОЛГЕ

Вниз по матушке по Во... по Волге,
По широкому раздолью,

По широкому раздолью... раздолью,
Поднималась непогода.

Поднималась непого... погода,
Погодушка немалая,

Погодушка немала... немалая,
Немалая, волновая.

Немалая, волнова... волновая,
Ничего в волнах не видно.

Ничего в волнах не ви... не видно,
Одна лодочка чернеет,

Одна лодочка черне... чернеет,
Только паруса белеют.

Только паруса беле... белеют,
На гребцах шляпы чернеют.

На гребцах шляпы черне... чернеют,
Сам хозяин во наряде,

Сам хозяин во наря... наряде,
В черном бархатном кафтане,

Уж как взговорит хозя... хозяин:
"Нуте, грянемте, ребята,

Нуте, грянемте, ребя... ребята,
Вниз по матушке по Волге.

Вниз по матушке по Во... по Волге,
Проворачивай, ребята,

Проворачивай, ребя... ребята,
Ко крутому бережочку!"




Такун Ф. И. Славянский базар. – М.: «Современная музыка», 2005

Песня неоднократно переделывалась, в том числе и в революционных целях: см. "Из-за матушки за Волги" Н. Огарева (1862), "Вниз по матушке по Волге" К. Федотова (1905-1906) - о баррикадных боях на Пресне. Неизменно использовалась в народной "разбойничьей" драме - см. драмы "Лодка", "Черный Ворон", "Шайка разбойников", "Ермак".


ВАРИАНТЫ (3)

1. Вниз по матушке по Волге...

Вниз по матушке по Волге
По широкому раздолью,

По широкому раздолью
Поднималась бурь-погода.

Поднималась бурь-погода,
Погодушка немалая,

Погодушка немалая,
Немалая волновая,

Немалая волновая.
Ничего в волнах не видно,

Ничего в волнах не видно,
Только видно красну лодку,

Только видно красну лодку.
Красна лодочка краснеется,

Красна лодочка краснеется,
На гребцах шляпы чернеются,

На гребцах шляпы чернеются.
Сам хозяин во наряде,

Сам хозяин во наряде,
Во коричневом халате,

Во коричневом халате.
«Вы притроньте-ка, ребята,

Вы притроньте-ка, ребята,
Ко крутому бережочку,

Ко крутому бережочку,
Ко желтому ко песочку».

Русские народные песни. / Сост. и вводн. тексты В. В. Варгановой. – М.: Правда, 1988, раздел «Песни разбойников и о разбойниках»

2. Вниз по матушке по Волге...

Вниз по матушке по Волге,
По широкому раздолью,
Разыгралася погода,
Погодушка верховая,
Верховая, волновая;
Ничего в волнах не видно,
Одна лодочка чернеет,
Только парусы белеют,
На гребцах шляпы чернеют,
На корме сидит хозяин,
Сам хозяин во наряде,
В черном бархатном кафтане.
Уж как взговорит хозяин:
«Ну-те грянемте, ребята,
Вниз по матушке по Волге,
Проворачивай, ребята,
Ко крутому бережочку».

«Великорусские народные песни». Изданы проф. А. И. Соболевским, т. I — VII, Спб. 1895—1902, IV, № 558

Русские песни / Сост. проф. Ив. Н. Розанов. М., Гослитиздат, 1952

3. Вниз по матушке, по Волге...

Вниз по матушке, по Волге,
По широкому было раздо — ой, раздолью.
Разыгралась с гор
Ой, пого — ой, погода,
Погодушка дует не малая,
Ой, не мала, не малая,
Не малая дует волнова — ой, волновая.
Ничего в волнах было не видно, не видно,
Только видна лодочка,
Чернеет, чернеет,
Паруса на лодочке
Белеют, белеют.

Записана от Антоновой Е. Н., 1911 г. р., г. Талгар, в 1976 г. Песня неизвестного автора XVIII-XIX в. См.: «Песни и романсы русских поэтов», серия «Библиотека поэта», М.-Л., 1965, № 118. Здесь лишь первая строфа песни.

Багизбаева М. М. Фольклор семиреченских казаков. Часть 2. Алма-Ата: «Мектеп», 1979, № 277
