Примітки: Акорди подано спрощено. Ориґінальна тональність: 1) за джерелом [1] - "Fm"; 2) за джерелом [2] - "Ebm". Підібрано на слух за аудіо-джерелами [1] і [2].

* Тут за джерелом [1] у 1-му куплеті за другим разом замість поданого тут тексту співається:
Вечорів оксамити,
Мов щастя прибій...
Як тебе не любити,
Києве мій!

** Ймовірно, тут: "канни" - множина від слова "канна". Канна - трав'яниста рослина (або її квітка) родини каннових.

*** Тут за джерелом [1] повторюється лише мелодія без тексту.

**** При останньому повторі тут замітсь акорда "Em" звучить "Em6".

Деякі акорди:
Am/F# [2 0 2 2 1 0]
Em* [3 2 2 0 0 0] або [x 2 2 0 0 3]
C6 [x 3 x 2 1 3]
Em6 [0 2 2 0 2 0]
