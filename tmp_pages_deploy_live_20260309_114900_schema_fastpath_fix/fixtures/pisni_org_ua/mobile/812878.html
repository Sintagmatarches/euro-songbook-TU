Не плачте не журіться

Сл: невідомий  
Муз: невідомий
Вик: Тарас Чубай, Плач Єремії
Вступ:  C A#*  (4)

     C          A#*        C 	G   C
Не плачте не журіться ви хлопці юнаки
    C          A#*       C     G     C
бо незабаром прийде ще бранка на дівки, -
    F                     C
розберем всіх до нага, як Еви у раю    |
   G                   C     G  C      | (2)
і так їх попровадим перед комісію.     |

Програш:  F C    (2)
          C A#*  (2)

Котра дівчина гарна і легка мов той птах,
то та буде служити напевно в літаках,
якщо хто буде важить сто кіля із дівчат, *  | (2)
то та піде служити до танків і гармат.      |

Котра дівчина гарна, а не впорядку все,
то та дістане книжку, а в ній велике "Ц".
розберем всіх до нага, як Еви у раю       | (3)
і так їх попровадим перед комісію.        |


* В джерелі:
Котра дівчина важить сто кіля із дівчат.

Акорд A#* [x 1 0 3 3 x] - можна не брати.
