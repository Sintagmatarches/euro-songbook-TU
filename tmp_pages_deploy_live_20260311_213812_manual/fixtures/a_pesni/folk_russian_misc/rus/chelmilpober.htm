ШЕЛ МИЛЫЙ ПО БЕРЕЖКУ

Шел милый по бережку,
Шел милый по крутому бережку шажком,
Переход себе искал.

Отломил он жердочку,
Отломил он тонкую жердочку над водой,
Сам по жердочке пошел.

Жердочка погнулася,
Шапочка стряхнулася с золотых волос,
Знать, мил потонул.

Увидала его девица,
Увидала красна девка из окна,
Из высока терема.

Вышла девка в сенечки,
Вышла девка в новые сени дубовые,
Сама вёдерцы брала.

Брала девка вёдерцы,
Брала девка новые вёдра зелены,
- Сама по воду пошла.

Почерпнула девка вёдерцы,
Почерпнула девка новые вёдра зелены,
Сама на гору пошла.

Становила девка вёдерцы,
Становила девка новые вёдра зелены,
Сама к лодочке пошла.




Русские народные песни. Песенник. Выпуск 9. Составитель Юрий Зацарный. М., «Советский композитор», 1990.
